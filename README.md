## Introduction
A very fun isometric tycoon factory game, where you can build a factory on your land and expand the
land! This game is still in early stages

## Quickstart
```sh
$ make
$ ./bin/app
```

## Controls
| Key        | Action                 |
| ---------- | ---------------------- |
| W          | Move cursor up         |
| A          | Move cursor left       |
| S          | Move cursor down       |
| D          | Move cursor right      |
| E          | Placing mode           |
| Backspace  | Deleting mode          |
| Enter      | Place/Delete           |
| ESC        | Exit the current mode  |
| Q          | Quit the game          |
